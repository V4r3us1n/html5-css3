::before & ::after

Esses pseudo-elementos permitem com que você adicione elementos ao seu HTML diretamente pelo CSS e, como você viu, pseudo-elementos possuem :: antes de seus nomes. Veja um exemplo de código:


            div.teste-before {
                width: 250px;
                margin: 10px auto;
                padding: 10px;
                background-color: rgb(73, 0, 0);
            }
            
            div.teste-before::before {
                display: block;
                height: 30px;
                width: 10px;
                background-color: rgb(0, 0, 36);
            }

Notou? O que foi escrito no ::before não apareceu, isso ocorre porque pseudo-elementos requerem que você use o atributo "content" para definir o que o conteúdo dentro do ::before vai ser, em nosso caso vamos usar apenas '' já que não queremos nenhum conteúdo.

Caso você inspecione o elemento usando o Chrome você vai ver que esses pseudo-elementos não estão antes/depois a div, mas sim dentro dela. Isso significa que elementos que não possuem tag de fechamento, como input, não podem utilizar pseudo-elementos

Como e Onde Posso Usá-los?

Para sinalizar cmapos obrigatórios em formulários, podemos usar o ::after, aí é só colocar uma classe em todos os campos obrigatórios e utilizá-lo nessa classe para adicionar o " * " após o nome do campo.

Pode ser usado para fazer bordas animadas :D

OBSERVAÇÃO: cada elemento da sua página pode ter apenas um ::before e ::after.
