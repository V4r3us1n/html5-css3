Sombras nas caixas

São muito úteis para dar a sensção de volume, que as caixas realmente estão ali. Veja um exemplo de código: 


            <style>
                body {
                    font: 1em Arial, Helvetica, sans-serif;
                    background-color: peachpuff;
                }
                article#caixa {
                    background-color: cyan;
                    width: 400px;
                    margin: auto;
                    padding: 1px 10px;
                    /* sombra aqui */
                }
            </style>

Usando sombras

Vamos aprender a usar sombras com esse exemplo simples.

No momento só temos um <article> com um breve conteúdo. As configurações deixaram ele do jeito acima. Para colocar a sombra vamos adicionar uma declaração especial: 


            box-shadow: 3px 5px 4px black;

Veja o que acontece agora lá em cima.

Ao adicionar a propriedade box-shadow tem quatro valores, devendo seguir a seguinte ordem:

  1. Deslocamento horizontal (h-offset): quanto a seombra vai andar para o lado direito (é possível usar valores nagativos também).
  2. Deslocamento vertical (v-offset): quanto a sombra vai andar para baixo (também é possível usar valores negativos).
  3. Embaçamento (blur): quanto a sombra vai se espalhar pelo fundo.
  4. Cor (color): cor da sombra, é possível usar transparência também.

OBS: Não exagere no uso de sombras, podem deixar o efeito visual pesado. Evite também sombras coloridas, além disso use cores rgba() para obter uma transparência que cause efeitos mais suaves.

Bordas decoradas

As bordas da caixa não precisam ser sempre iguais. É possível colocar detalhes especiais: 


            border-radius: 10px;
