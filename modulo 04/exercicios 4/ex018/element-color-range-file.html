Elementos Color, Range e File em HTML

Color

O input to tipo color serve para que você posso selecionar uma cor em um sistema de cor (HSL, RGB, Hexadecimal), os valores serão enviados de acordo com o sistema de cor escolhido e o valor selecionado, ex:

Cor vermelha:

  • RGB: 255, 0, 0
  • HSL: 0, 100%, 50%
  • Hex: #FF0000

Independente do sistema de cor que for utilizado, o valor representativo enviado será no sistema hexadecimal, no exemplo acima seria enviado %23ff0000, o % representa a # usado.

Caso queira deixar uma cor predefinida assim q o site carrega, basta colocar no input um value com o valor hexadecimal da cor

Range

Em inputs do tipo range é necessário algumas informações a mais, mas ele é basicamente uma barra q vc pode subir o valor dela (puxando a bolinha para a direita) ou descer o valor (puxando para a esquerda). Se colocarmos no valor mínimo e enviarmos, esse valor será representado por 0 na URL, porém se colocarmos ele no máximo ele será 100.

Esse tipo de input é usado normalmente para avaliar coisas, dando uma nota.

Ao configurar o range você pode colocar valor mínimo e máximo ao mesmo, o valor que ele vêm carregado é o 3, que corresponde a metade. Com os valores utilizados no exemplo é possível notar que independente da direção em que a barra é buxado seu movimento é mais rígido, diferente de antes.

Se você quer que ele comece em um valor predefinido basta adicionar um value com o valor que você deseja.

File

Quando utilizar controles de envio de arquivo, como inputs do tipo file, você não pode utilizar no formulário o method="get" (aguenta só até 3000 bytes, menor que muitas imagens), você deve utilizar o method="post" para que os arquivos sejam realmente enviados.

Dessa forma a imagem será enviado também em forma de dados para o servidor.

Avisos!

Vale a pena ressaltar que esses 2 inputs podem ser apresentados de forma diferente em outras navegadores, podendo virar até mesmo caixas de texto.

Por isso lembre de testar eles em múltiplos navegadores antes de lançar seu site.
