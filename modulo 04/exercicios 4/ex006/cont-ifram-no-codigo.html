Conteúdo no iframe por Código

Na aula anterior fizemos um iframe que permitia com que exibíssemos diferentes conteúdos dentro dele através de links vínculados em um mesmo target.

  • Primeira Página
  • Segunda Página
  • Terceira Página

No entanto, quando iniciamos a página o iframe começa vazio, e isso não é muito atrativo.

Poderíamos colocar um dos arquivos para ficar lá pronto para ser alterado se necessário, apenas colocando no src do iframe o link de uma das páginas criadas, mas vamos fazer outra coisa.

Existe um parâmetro chamado srcdoc, esse parâmetro permite você colocar dentro dele HTML, veja:


            <iframe id="tela" name="frame" srcdoc="<h1>Escolha uma das opções acima</h1>">

O resultado será o seguinte:

Para adicionar parâgrafos ou mais conteúdo você terá de colocar eles na mesma linha, ex:

<h1>Oi</h1><p>eu sou o goku</p>

Se você precisar adicionar uma imagens ou algo que você precise usar aspas, lembre-se de usar aspas diferentes dentro do srcdoc, use aspas simples dentro dele.

Esse conteúdo que colocamos só poderá ser exibido dnv se a página for recarregada.

Sendo sincero, para coisas mais complexas vale mais a pena Criar um arquivo separado do que usar o srcdoc para exibir uma imagem no iframe.
