Listas com HTML5

Caso você queira deixar um comentário em seu código basta usar a tag <!-- -->.
Caso você queira alterar várias tags ao mesmo tempo, basta você clicar em uma das tags e segurar o "alt", a partir do momento que você clicou no "alt" basta mantê-lo pressionado enquanto vai clicando nas outras tags que quer alterar, isso fará com que você consiga alterar várias linhas ao mesmo tempo.

O poder das listas

As listas são muito importantes, pois elas simplificam e sintetizam o conteúdo a ser passado, além disso elas fazem a demarcação de cada item para facilitar a vizualização.

Pensando nisso, a HTML disponibiliza vários tipos de listas, para deixar nosso conteúdo mais claro e eficiente.


Listas Ordenadas

São as listas ondem a ordem dos itens é muito importante, como um passo-a-passo para fazer um bolo.

Para criar uma ordered list, deve-se utilizar a tag <ol> para delimitar a lista e a tag <li> (list item) para identificar cada item da lista. Exemplo:

  1. 2 colheres de açúcar
  2. 1 colher de azeite
  3. 3 ovos
  4. 1/2 xícara de leite
  5. 1 xícara de farinha de trigo

Vale ressaltar que a tag <li> é de fechamento opcional no HTML5, diferente da tag<ol>.

A tag <ol> possui um parâmetro type, onde é possível configurar o tipo de marcador de lista a ser utilizado. As opções possíveis para esse parâmetro são:

  1. 1 - Valor padrão. Listas númeradas: 1, 2, 3, ...
  2. A - Listas alfabéticas em maiúsculas: A, B, C, ...
  3. a - Listas alfabéticas em minúsculas: a, b, c, ...
  4. I - Listas com algoritmos romanos maiúsculos: I, II, III, ...
  5. i - Listas com algoritmos romanos minúsculos: i, ii, iii, ...

Também é possível indicar o início da contagem usando o parâmetro "start".
Ex: A tag <ol type="I" start = "5"> vai fazer com que alista seja numerada a partir do número "5" em romano.

Listas não Ordenadas

São usadas quando a ordem dos itens não influencia no significado da lista. É uma ótima maneira de organizar os itens que não apresentam, necerrariamente, uma classificação.

Para criar uma unordered list basta utilizar a tag <ul> para delimitar a lista e a tag <li> para criar cada um de seus itens. Ex:

  • Carro
  • PC Gamer
  • Bujão de gás cromado
  • Waifu

Nesse tipo de lista o marcador comum é a bolinha totalmente preenchida (circle), mas é possível configurar a a propriedade type dessa tag. É possível escolher os seguintes valores:

  • disc - Padrão, bola totalmente preta
  • circle - Uma cola com a borda preta mas sem preenchimento
  • square - Um pequeno quadrado totalmente preto

Listas Mistas

É possível criar listas mistas, para isso basta configurar uma lista dentro de outra lista. Ex:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
  3. Item 3
  4. Item 4

É possível notar que os itens da lista interna estão deslocados para a direita em relação a lista externa.

Lista de Definições

É basicamente um dicionário, nela se colocam os termos e suas descrições, ela não tem demarcadores mas ainda sim é bem útil!

Para fazer uma definition list basta usar a tag <dl>. Nela cada termo é um <dt> (definition term) e cada descrição é um <dd> (definition description). Assim como o item list essas duas últimas tags possuem fechamento opcional. Ex:

HTML
Linguagem de marcação utilizada para criar o conteúdo de sites.
CSS
Linguagem de marcação para a especificação de estilos em sites.
JavaScript
Linguagem de programação para criar interatividade em sites.

Cada navegador pode apresentar essa lista de forma diferente, mas o que importa é a relação entre Termo e Descricição, que o significado dessa lista.
