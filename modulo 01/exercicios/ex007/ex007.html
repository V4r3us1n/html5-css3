O que é semântica?

Colocamos o arquivo como "index.html", porque index significa índice, que é a página principal do projeto, e como só usamos uma página ela sempre será index.


É ela a principal diferença entre o HTML4 e HTML5.

Segundo o pai-dos-burros,Michaelis, que tem uma versão grátis online:
"Semântica é o significado dos vocábulos, por oposição à sua forma".

No HTML4 eram usados tags para colocar palavras em negrito e sublinhado,
mas isso era usado só para determinar um formato visual para chamar sua atenção à algo.

No HTML5 chegou um novo conceito, que busca valorizar a semântica.
Logo, suas tags tentam levar um significado embutido muito forte. Aqui está um exemplo:
Juvenal era um sujeito de muita sorte. E já começou de pequeno onde morou na

Rua 21 Fernando de Noronha
, local muito bem localizado, próximo ao Centro da cidade.

Note que no exemplo acima a tag <address> foi usada para dar significado à palavra destacada, mostrando que se trata de um endereço. Em um celular essa tag pode até sugerir que você veja o mapa do local, entre outras coisas.

Caso você queira que um endereço apareça na forma de um texto sublinhado, recomendo que use CSS para isso, pois o HTML serve para dar sentido ao conteúdo.
Tanto que a W3C anda dando muito valor em adicionar tags que tenham mais significado e a tirar aquelas que focam no efeito visual (forma) de apresentação, deixando a apresentação gráfica para as CSS.

Caso você queira colocar uma frase dentro de uma tag, selecione ele e aperte "crtl + shift + p", digitar "abb" e selecionar ">Emmet: Wrap with abbreviation" (Envelope com a abreviatura), aí basta você digitar a tag sem os colchetes angulares.
Exemplo: essa frase está em itálico, diferentemente da frase acima que está em negrito.

Abandono de tags antigas e o surgimento de novas

Caso você queira basta pesquisar "obsolete tags" no w3.org. Se você estiver usando alguma tag em seus estudos e ela aparecer em vermelho, é bom que você não use ela mais, pois ela é considerada obsoleta e a qualquer momento vai parar de funcionar. O HTML5 é usado para dar significado, deixe a forma para a CSS e as animações para o JS.

Tags obsoletas vêm sendo esquecidas e deletadas, já que existem melhores. No caso do <b> que deixa a palavra em negrito é recomendado que use a tag <strong>, que dá o sentido de força/potência dentro da frase.

Para usar o negrito e o itálico, recomendo que use as seguintes tags:
A tag <strong> que dá o sentido de força dentro da frase.
A tag <em> significa que queremos dar ênfase ao termo.
É possível fazer o mesmo que as canetas marca texto em seu site, elas servem para marcar um trecho que você considere muito valioso. Para fazer essa marcação usamos a tag <mark>.


Aumentar e diminuir o tamanho do texto

A tag <big> que deixa o texto maior está sendo desvalorizada pelo HTML5, mas a tag <small> que deixa o texto menor ainda está de pé.
Também pode ser utilizado a tag <del> para selecionar uma parte do texto que não deve ser considerada pelo leitor.
O texto inserido é o oposto, usando a tag <ins>, serve para dizer que o texto está ali, deve ser lido e você deve prestar atenção nele.


Texto sobrescrito e subscrito

Parafazer textos sobrescritos e subscrito basta usar a tag <sup> ou <sub>. Não sabe o que são, tome aqui um exemplo deles:
Texto sobrescrito: x2;
Texto subscrito: H2O;


Colocando códigos no texto

Existem muitos código-fonte compartilhados em diversas linguagens. Para isso entra a tag <code> da HTML onde ela delimita o código, você deve colocar o código entre essa tag, ela serve para indicar ao navegador que aquele trecho se trata de um código de computador. Ela também causa um efeito visual, deixando as letras mono-espaçadas, facilitando a leitura do código. Exemplo de Código em Python:

num = int(input('Digite um número')) if num % 2 ==0: print(f'O número {num} é par') else: print(f'O número {num} é impar') print('FIM DO PROGRAMA')

Para manter o texto da mesma forma que ele foi digitado utilizados a tag <pre> antes da tag <code>. Observe:


    num = int(input('Digite um número'))
    if num % 2 ==0:
        print(f'O número {num} é par')
    else:
        print(f'O número {num} é impar')
    print('FIM DO PROGRAMA')

Como você viu acima é possível colocar uma tag dentro da outra e assim fazendo com que as duas ajam simultâneamente. Outro exemplo: escrever algo em negrito e em itálico ao mesmo tempo!


Citações

Para fazer citações basta usar a tag <blockquote>, oq vc escrever dentro dessa tag automaticamente vai ganhar um espaçamento, além disso você também pode colocar o endereço de onde pegou a citação, para isso basta colocar o parâmetro"cite" dentro da tag. Exemplo:

Omori nunca saiu do seu quarto (White Space) e toda sua aventura, lembranças e essas coisas é fruto da sua imaginação.

Abreviações

Sempre que você quiser escrever uma sigla, mas deixar claro ao usuário (e aos mecanismo de busca) o significado dela, basta usar a tag <abbr>. Exemplo:

Eu estou estudando HTML para criar sites.


Texto invertido

Quando você passar o mouse por cima da abreviação vai aparecer o significado dela.

Para fazer textos invertidos use a tag <bdo>, você pode usar o parâmetro "dir" para indicar uma das opções abaixo:
rtl = da direita para a esquerda;
ltr = da esquerda para a direita;
Exemplo:
Esse texto está escrito da direita para esquerda!
Esse texto está escrito da esquerda para a direita!
