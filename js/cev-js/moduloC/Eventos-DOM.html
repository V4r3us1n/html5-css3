Eventos DOM

DOM é um facilitador, que existe em seu navegador, para dar acesso aos elementos diretamente para o JS.

Interaja

Vamos usar como exemplo essa div, um "Evento" é tudo aquilo que pode acontecer com essa div. Os mais comuns são eventos de mouse, quando o mouse entrar na div vai disparar um evento chamado "mouseenter", além disso você poe também continuar movendo o mouse dentro da div, disparando várias vezes o evento "mousemove". Se você clicar e segurar pode disparar o evento do "mousedown" e no momento que você soltar o botão do mouse você dispara o "mouseup". Além disso também tem o movimento do click inteiro, clicar e solter rapidamente, que é o evento de "click". E por fim, assim que você tira o mouse de dentro da div você dispara o evento do "mouseout".

Todos os eventos mencionados anteriormente acontecem só com o uso do mouse, existe vários eventos dentro do JS. Pesquise por JS DOM Events para mais informações.

Mas para disparar um evento, para tratar um evento, você tem que saber o que é uma função. Uma Função é um conjunto de linhas que vão ser executadas só quando o evento ocorrer.

Pense da seguinte forma, você programou 10 linhas, o que chamamos de bloco, diferentemente do que fizemos até agora esse bloco só será executado quando algo específico ocorrer. Para essas 10 linhas serem executadas apenas quando o evento ocorrer você tem que colocar elas dentro de um bloco, que é delimitado por um par de {}. Você tem que nomear esse bloco como uma Function antes do bloco. No JS existem funções anônimas que são aquelas que não tem um nome, mas para o método funcionar a função precisa de um nome. Geralmente o nome da são ações, você coloca o nome da "ação()". Ficando algo assim: 


        function ação() {
            Linhas de Código :)
        }

Além disso, opcionalmente, dentro dos parênteses você pode colocar alguns Parâmetros, que serão abordados depois.

Vamos fazer um exemplo de evento de click. Lembrando que os eventos podem ser disparados tanto na parte HTML quanto na parte do JS. Veja na DIV que criamos anteriormente como isso é feito no HTML.

Fazendo isso usando o HTML ficaria algo assim: 


        <html>
            <head></head>
            <body>
                <div id="area" onclick="clicar()" onmouseenter="entrar()" onmouseout="sair()">
                    <p>Interaja</p>
                </div>
                <script>
                    //Nesse exemplo foi criado a variável fora das funções para ela ser global, mas isso pode acabar gerando uns problemas com memório por isso não é muito recomendado.

                    var a = document.querySelector('div#area');

                    function clicar() {
                        //Essa função só será chamado quando o "clicar" for ativado
                        a.innerText = 'Clicou XD';
                        a.style.background = 'black';
                    }

                    function entrar() {
                        //Essa função vai ser chamada quando "passar o mouse" sobre a div
                        a.innerText = 'Entrou :D';
                    }

                    function sair() {
                        //Essa função vai ser chamada quando "tirar o mouse" de dentro da div
                        a.innerText = 'Saiu D:';
                        a.style.background = 'rgb(255, 0, 242)';
                    }
                </script>
            </body>
        </html>

Você tembém pode disparar os eventos usando "listeners" que ficam dentro do JS. Ex: 


        //Nesse exemplo foi criado a variável fora das funções para ela ser global, mas isso pode acabar gerando uns problemas com memória por isso não é muito recomendado.

        var a = document.querySelector('div#area');

        //Tem que colocar um Listener para cada Evento!

        a.addEventListener('click', clicar)
        a.addEventListener('mouseenter', entrar)
        a.addEventListener('mouseout', sair)

        function clicar() {
            //Essa função só será chamado quando o "clicar" for ativado
            a.innerText = 'Clicou XD';
            a.style.background = 'black';
        }

        function entrar() {
            //Essa função vai ser chamada quando "passar o mouse" sobre a div
            a.innerText = 'Entrou :D';
        }

        function sair() {
            //Essa função vai ser chamada quando "tirar o mouse" de dentro da div
            a.innerText = 'Saiu D:';
            a.style.background = 'rgb(255, 0, 242)';
        }

Preste bastante atenção em seu código, pois o JS não avisa que ocorreu um erro, se algo estiver errado ele não avisa. Como achar esse erro então? Bom, basta dar um "inspecionar" na página, na parte superior do inspecionar vai ser notificado que houve um erro, em 'Console' ele vai mostrar o tipo de erro e vai apontar onde se encontra o erro, aí basta você corrigí-lo.
