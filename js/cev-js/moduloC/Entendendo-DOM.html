Entendendo o DOM

No módulo C nós vamos entender o que é o Document Object Model (DOM).

Ele é a vertente web do JS para o qual a linguagemf oi criada, se trata de uma ferramenta presente em seu navegador que ajuda no tratamento de componentes visuais (na criação deles).

Além disso também vamos aprender a criar uma Árvore DOM do seu site, conhecendo a estrutura DOM de seu site.

Além de conhecer tal estrutura você também conseguirá Manipular a Estrutura DOM de qualquer site.
