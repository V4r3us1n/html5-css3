O que é DOM?

Como já citado anteriormente, DOM é Document Object Model (Modelo de Objetos para Documentos), se trata de um conjunto objetos dentro do seu navegador que vai te dar acesso aos componentes internos de seu website.

Ele só funciona dentro de navegadores.

Árvore DOM

Raíz (window), começa a partir dela, tudo no JS está dentro de um objeto chamado "window" (janela).

Dentro do window existem vários outros objetos, como:

  • location : diz a localização do seu site, url, página atual, página anterior, etc
  • document : documento atual
  • history : guarda informações como "de onde você veio", "para onde você vai", etc. Facilita a navegação no seu site.

Dentro do "document" existe outro objeto muito importante que é o "html".

Dentro do "html" temos 2 objetos/child: o "head" e "body".

Seria algo assim: 


            window:
                location
                document
                    html
                        head
                            meta
                            title
                        body
                            h1
                            p
                            p
                                strong
                            div
                                p              
                history

Os objetos que estão dentro de outros são chamados "child" e os objetos que abrigam eles são os "parent"

Todos os objetos mostrados no exemplo anterior são elementos, você pode acessar eles com vários metódos, alguns deles são:

  • por Marca (tag)
  • por ID
  • por Nome
  • por Classe
  • por Seletor (CSS)

Selecionando elementos por Marca

basta você usar o seguinte comando:


            window.document.getElementsByTagName('p')
            //Pode ocultar o "window"

Com ele você consegue selecionar mais de um objeto, como parágrafos, não existe só um.

Selecionando elementos por ID

QUando seu site é mais desenvolvido e voce^tem que selecionar elementos únicos, você pode usar essa seleção. Basta usar o seguinte comando:


            documento.getElementById()

Selecionando elementos por Nome

Se você tem a propriedade "Name" você pode usar o seguinte coamndo para selecionar:


            document.getElementsByName()

Percebeu que o "Elements" está no plural? Mostra que você tem que usar o colchetes quando quiser selecionar algo específico.

Selecionando elementos por Classe

Entendo os outros você entende esse também, então use o seguinte comando:


            document.getElementsByClassName()

Selecionando elementos por Seletor


            querySelector(div#msg)
            ou
            querySelectorAll()

Você usa basicamente a síntaxe do CSS, colocar o nome do elemento: div e o id/classe (identificador) #msg
