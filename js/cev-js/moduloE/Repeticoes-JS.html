Repetições em JavaScript

é nesse ponto que muita gente entra em desespero, nesse módulo nós vamos ver:

  • Estruturas de Repetição com Teste Lógico no ínicio (while)

Essas são as 3 estruturas de repetição básicas das linguagens de programação.

Repetições (Parte 1)

As estruturas de repetições começam da mesma forma que uma estrutura de condição, ou seja, com um teste lógico que pode ser Verdadeiro ou Falso. No entanto, ele se comporta de forma diferente, se o teste lógico for Verdadeiro ele vai executar o bloco e, ao invés de continuar a sequência, ele vai voltar para refazer o teste, isso vai continuar (entra em loop) até que o resultado seja Falso.

Enquanto a condição for verdade o loop (laço) será mantido, quando ela for falsa o loop será quebrado.

A primeira estrutura que vamos testar é o while, veja: 


        while (condição) {
            [bloco]
        }

Vamos para um exemplo mais prático: 


        function comerPizza() {
            while (temFatia()) {
                comerFatia();
            }
        }

Vamos fazer alguns testes (Repeticao-p1.js)

Como pode ver o while é possui teste lógico no início, ou seja, primeiro ele realiza o teste lógico e depois ele executa o bloco, para aí ele entrar em loop, ou não.

Existe também uma estrutura oposta a ela, que primeiro executa o bloco e depois realiza o teste lógico, para então entrar em loop ou não. Chamamos essa estrutura de repetição de do while, e funciona da seguinte forma: 


        do {
            [bloco]
        } while (condição)
