Estrutura FOR

Ela é representada por um "hexagono irregular", nele é feito a Inicialização, Teste Lógico e um Incremento. Na primeira execução dele são feitos a "Inicialização" e o "Teste Lógico", se o teste lógico for verdadeiro ele vai executar o bloco e vai iniciar o loop, no entanto, dessa vez é realizado também o Incremento (aumentando o valor do contador).

Esse loop será repetido até que o Teste Lógico seja falso!

Veja como é escrito essa estrutura em JS: 


        for (inicio; teste; incremento) {
            [bloco]
        }

Vamos para um exemplo mais prático: 


        //Veja como é feito no while:

        var c = 1;
        while ( c <= 4) {
            [bloco]
            c++;
        }

        //Veja isso com o for:

        for (var c = 1; c <= 10; c++) {
            [bloco]
        }

Eles fazem a mesma coisa, no entanto muito programadores mais experientes preferem usar o "FOR" para grande parte das vezes quando você sabe o limite de execuções a ser feito!
