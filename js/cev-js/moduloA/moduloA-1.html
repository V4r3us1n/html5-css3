Client x Server

Nós somos clientes e precisamos de servidores para acessar o que queremos com nossos dispositivos. Um cliente faz requisições de arquivos e eles são enviados pelos servidores. O JS é usado para o lado dos clientes, mas ele também pode ser usado no server-side.

Um website é criado principalmente por 3 tecnologias: o HTML, CSS e JavaScript.

  • HTML: é responsável pelo conteúdo do site.Faz textos, marcações e adição de imagens e vídeos.
  • CSS: é responsável por estilizar o conteúdo, tornando-o mais bonito.
  • JS: é responsável pela integração e interação entre o site e os usuários.

Esses 3 são as tecnologias principais que compõem o Client-Side.

O JS é usado por muitas empresas, como Google, facebook, netflix, etc.
