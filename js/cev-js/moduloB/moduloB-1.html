Variáveis

Em um estacionamento existem vagas para carros, motos e caminhões, cada um armazenando um tipo diferente de automóvel. Além disso, para você não esquecer onde estacionou, cada vaga tem um nome.

Para colocar um carro em uma vaga podemos dizer que: vaga a1 = carro1.

Ou seja, a vaga 1 recebeu o carro 1. Mas e se chegar outro carro que quer estacionar na mesma vaga em que o carro 1, tem como ele estacionar lá?

A resposta é não, para ele estacionar o carro 1 tem que sair da vaga, e é assim que funciona nas variáveis.

E se você não quiser que nenhum carro ocupe aquela vaga? Para isso podemos usar um comando como a1 = null. Isso tira o carro que está na vaga.

Assim como as vagas, as variáveis também são espaços que armazenam coisas e elas também precisam de um identificador único para cada uma. Você consegue atribuir valores a essas variáveis e, assim como os carros ficam nas vagas, eles irão ficar aramazenados nela até que outro valor os substitua.

Da mesma forma que no estacionamento existem vagas de tamanhos diferentes, também existem variáveis de tamanhos diferentes, mas ao invés de carros, caminhos e motos, elas podem armazenar números inteiros, reias, strings, valores booleanos, etc.

Para armazenar strings você tem que usar um símbolo para mostrar que o valor é uma string, para isso podemos usar: "exemplo 1", 'exemplo 2' e `exemplo 3`iferenças entre eles no JS, mas faleremos sobre isso depois.

O nome de cada variável é o que chamamos de identificador, existem algumas regras para os identificadores no JS:

  • Podem começar com letra, $ ou _;
  • Não podem começar com números;
  • É possível usar letras ou números;
  • Épossível usar acentos e símbolos;
  • Não podem conter espaços, subtituímos ele por "_";
  • Não podem ser palavras reservadas, palavras que o JS usa em seus comandos.

Dicas para escolher Nomes de Identificadores

  • Maiúsculas e minúsculas fazem diferença. Você tem que escrever o nome variável da mesma forma que você a nomeou anteriormente;
  • Tente escolher nomes coerentes para as variáveis, algo que faça sentido;
  • Evite se tornar um 'programador alfabeto' ou um 'programador contador', não use apenas letrar individuais ou números (N1, N2, N3...) para identificar variáveis, vai gerar confusão;

Tipos Principais de Variáveis no JS

Diferente de outras linguagens, o JS não diferencia Números Inteiros de Números Reias (float), todos os números se encaixam como number.

Qualquer informação armazenado entre aspas, mostradas anteriormente, são classificados como String.

E as informações de True or False são Valores Booleanos;

Existem outros tipos de dados, sendo eles:

  • number
    • Infinity
    • NaN (Not a Number)
  • string
  • boolean
  • null
  • undefined
  • object
    • array
  • function

DICA: para descobrir o tipo de uma informação você precisa usar o typeof.
