No caso abaixo os dados inseridos na variável são tratados como Strings, e quando você "soma" os valores você na verdade são concatenados. 


        var n1 = prompt('Digite um número: ');
        var n2 = prompt('Digite outro número: ');
        var s = n1 + n2;
        alert('A soma entre os valores é ' + s);

Para que o JS entenda que o operador "+" servirá para somar, e não concatenar, você deve indicar para ele que os valores utilizados na operação são números e não strings.

Vamos ter que realizar uma conversão de tipos de "String > Número". Existem 3 formas de se fazer isso, sendo elas:

  • Number.paseInt(n) ou parseInt(n) [Serve para números inteiros]
  • Number.parseFloat(n) ou parseFloat(n) [Serve para números quebrados]
  • Number(n) [Usando apenas isso o JS vai descobrir por conta própria se o valor é Inteiro ou Real]

Por mais que a última opção seja "melhor" do que as outras, elas também são necessárias, você pode acabar encontrando situações onde os valores tem que ser, obrigatoriamente, Inteiros ou Reais apenas.

Para fazer a conversão de número para String existem 2 opções:

  • String(n)
  • n.toString()

Toma aqui um exemplo dessas conversões: 


        var n1 = parseInt(prompt('Digite um número: '));
        var n2 = parseInt(prompt('Digite outro número: '));
        var s = n1 + n2;
        alert('A soma entre os valores é ' + s);

        var n3 = parseFloat(prompt('Digite outro número: '));
        var n4 = parseFloat(prompt('Digite outro número: '));
        var s2 = n3 + n4;
        alert('A soma entre ' + n3 + ' e ' + n4 + ' é ' + String(s2)); //Convertendo o valor de s2 para String

        var n5 = Number(prompt('Digite outro número: '));
        var n6 = Number(prompt('Digite outro número: '));
        var s3 = n5 + n6;
        alert(`A soma entre ${n5} e ${n6} é ${s3.toString()}`); //Convertendo o valor de s3 para String

Não precisa usar o ".toString()", só a variável já basta aqui.

IMPORTANTE: antes você tinha que usar o '+' para concatenar sempre que envolvia uma variável. Mas agora nas novas versões tem outro jeito: TemplateStrings 


        `Eu estou usando ${s}` //É obrigatório o uso de craz

Existem outras coisas que se pode fazer com Strings, como:

  • .length //quantos caracteres a string tem
  • .toUpperCase() //tudo para 'MAIÚSCULAS'
  • .toLowerCase() //tudo para 'minúsculas'
