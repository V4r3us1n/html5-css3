Usando o "prompt" permitimos que o usuário insira dados, mas como podemos armazenar esses dados?

Para armazenar o dado inserido basta você colocar o comando "prompt" dentro de uma variável. Dessa forma você poderá trazer essa informação sempre que necessário.
