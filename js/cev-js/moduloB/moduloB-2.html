Manipulação de Dados

Observe o seguinte código: 


        //Guardando uma informação
        prompt('Qual é seu nome?')

Ele solicita uma informação para o usuário, mas essa informação não vai apra lugar algum. Como podemos fazer para que essa informação fique armazenada em algum lugar??

Para armazenar essas informação basta você colocá-la em uma variável, ex: var nome = prompt('Qual é seu nome?')
OBS: você pode usar também, ao invés de var, o comando let ou const, há uma diferença entre eles que iremos ver mais pra frente então por enquanto use var

Dessa forma, sempre que você precisar trazer essa iformação à tona basta você usar o nome da variável que a armazenou.

Quero que o usuário digite 2 números, então: 


        var n1 = prompt('Digite um número: ');
        var n2 = prompt('Digite outro número: ');
        var s = n1 + n2;
        alert('A soma é ' + s);

Porquê ocorreu uma concatenação e não soma já que se tratam de números???
É porque em nenhum momento foi específicado para o JS que aquela informação se tratava de um número.
