Operadores do JS (Parte 2)

Vamos abordar agora os últimos 3 operadores, os operadores relacionais, lógicos e ternário.

Operadores Relacionais

Os operadores realacionais do JS são os da maioria das linguagens de programação, sendo eles:

  • > Maior que
  • < Menor que
  • >= Maior ou igual a
  • <= Menor ou igual a
  • == Igual a
  • != Diferente de

Vale citar que qualquer expressão que contenha um operador relacional vai gerar um valor booleano, Verdadeiro ou Falso. Ex:

  • 5 > 2 == True
  • 7 < 4 == False
  • 8 >= 8 == True
  • 9 < 7 + 1 == False
  • 5 == 5 == True
  • 4 != 4 == False

Lembrando, sempre que você tiver em uma expressão operadores relacionais e operadores aritméticos, primeiro serão realizados os operadores airtmáticos!

Também temos que citar os operadores relacionais conhecidos como "operadores de identidade". Ex: 


            5 == 5 (true)
            5 == '5' (true)

O que acontece aqui é que o JS avalia se eles tem a mesma grandeza, e diz que eles são iguais.

Para isso existe o operador de identidade ('===') ele testa se eles tem o mesmo valor e o mesmo tipo, ficando algo assim: 


            5 === '5' (false)
            5 === 5 (true)

Existem também o operador desigual restrito ('!=='), que não serve para avaliar o tipo, mas sim o valor interno deles, se forem iguais então é true. 


            5 !== 5 (false)
            5 !== '5' (true)

Agora vamos abordar os próximos operadores, que são os relacionais!

Operadores Lógicos

Os operadores lógicos são os seguintes:

  • ! Negação ('não' lógico)
  • && Conjunção ('e' lógico)
  • || Disjunção ('ou' lógico)

Quero uma caneta mas ela não(!) pode ser azul.

Quero uma caneta azul e(&&) uma caneta vermelha.

Quero uma caneta azul ou(||) uma caneta vermelha.

Vamos começar com o Negação !.

Nele você só pode colocar uma expressão que vai dar 'true' ou 'false', ele normalmente vêm junto com um operador aritmético (ex: != [não igual]).

Uma coisa que não é verdadeira é falsa e uma coisa que não é falsa é verdadeira.

Vamos para o segundo, a Conjunção &&

Ele é um operador binário, precisa de dois valores lógicos, um de cada lado. A expressão só será verdadeira se os dois valores lógicos forem atentidos (sejam verdadeiros).

Por último, mas não menos importante, a Disjunção ||.

Também é um operador binário, no entanto ele precisa que pelo menos um dos dois valores lógicos for atendido, se tornando falsa somente quando os dois forem falsos.

Quando se tem os 3 operadores lógicos em uma expressão a ordem é a seguinte (da esquerda para direita): ! > && > ||.

Veja este exemplo: 


            a > b && b % 2 == 0

Quando se tem operadores lógicos, relacionais e aritméticos na mesma expressão a ordem é a seguinte: aritméticos, relacionais e lógicos.

No exemplo acima a resposta é 'false', pq apenas um dos lados é verdadeiro. 


            alert('Estamos realizando uma seleção, responda as perguntas abaixo: ')
            idade = Number(prompt('Informe sua idade: '));
            estado = String(prompt('Informe a sigla do seu estado: [ex: RJ]'));
            salario = Number(prompt('Informe seu salário: '));
            sexo = String(prompt('Informe seu sexo: [M/F]'));

            if((idade >= 15 && idade <= 17) && (estado == 'RJ' || estado == 'SP') && (salario > 1500 && sexo != 'M')) {
                alert('Você passou na seleção!');
            } else {
                alert('Você não passou na seleção!');
            }   
            alert('Fim da pesquisa :)');

Operador Ternário

É composto por 2 símbolos que aparecem na mesma expressão, sendo eles:

  • ?
  • :

Ele se chama ternário porque tem 3 partes:


            teste ? true : false

Onde:

  • Bloco teste: é um teste lógico, um resultado que pode dar verdadeiro ou falso.
  • Bloco verdadeiro: é o que vai acontecer quanto o "Bloco teste" for verdadeiro.
  • Bloco falso: é o que vai acontecer quando o "Bloco teste" for falso.

Ex:


            nota1 = Number(prompt('Informe sua primeira nota: '));
            nota2 = Number(prompt('Informe sua segunda nota: '));
            media = (nota1 + nota2) / 2;
            alert(media >= 7.0 ? "Aprovado" : "Reprovado");

Outro exemplo para você entender:


            x = Number(prompt('Informe um valor: '));
            var res = x % 2 == 0 ? 5 : 9;
            alert(res);

Se o resto da divisão de 'x por 2' for 0 então o valor de "res" será '5', senão ele será '9'.
