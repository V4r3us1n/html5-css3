Operadores do JS (Parte 1)

Existem várias famílias de operadores no JS, mas vamos focar apenas nos seguintes:

  • Aritméticos
  • Atribuição
  • Relacionais
  • Lógicos
  • Ternário

Nessa priemira parte vamos abordar apenas 2, os operadores aritméticos e os de atribuição.

Operadores Aritméticos

VocÊ já deve conhecer o que são operadores aritméticos, então, sem mais delongas, aqui estão os operadores aritméticos usados no JS:

  • + Soma
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Resto da Divisão Inteira
  • ** Exponenciação (Potência)

Assim na matemática existe a precedência de operadores, quais operações se realizam primeiro. Caso você a ordem de precedência o código vai rodar, mas não vai fornecer o resultado esperado. O código só apresenta erro se houver um erro na sintaxe dele, ex: 5 + / 2.

Só porque o programa não apresentou erros não quer dizer que ele esteja certo, você pode ter cometido algum erro sintático.

Utilizando os "()" você consegue alterar a ordem de precedência, que será exibida a seguir:

  • ()
  • **
  • * / %
  • + -

Caso os operadores tenham a mesma posição na linha de precedência eles serão excutados da esquerda para a direita.

Operadores de Atribuição

E se eu quiser realizar o cálculo e armazenar ele? Como faço?

Para isso você usa os Operadores de Atribuição, também conhecidos como variáveis.

Lembrando que é possível você dar novos valores a uma variável que já esteja ocupada, por exemplo: 


            var n = 3;
            n = n + 4;
            alert(n);

No exemplo acima o n que antes possuía valor 3 passou a possuir o seu valor anterior + 4.

Também é possível simplicar essas auto-atribuições, que no caso seria assim: n += 4.

Lembrando que isso só pode ser usado para auto-atribuições, podendo ser usado com qualquer operador, não necessariamente a soma, e também é aceito pela maioria das linguagens.

Operadores de Incremento

Esse tipo de operador serve para simplificar ainda mais essas operações. Ex: 


            var x = 5;
            x = x + 1;
            x += 1;
            x ++;

Essa simplificação vai sempre adicionar, ou tirar, 1 do valor da variável. No entanto se atente a uma coisa, caso você use o x ++ o valor só será atribuída na próxima vez que você sumonar a variável. Se você que isso se aplique na hora você só troca a ordem para ++ x. Chamamos isso de Pós-Incremento (x ++) e Pré-Incremento (++ x).
