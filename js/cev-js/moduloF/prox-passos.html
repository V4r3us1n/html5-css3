Mas isso não é tudo, o JS é uma linguagem enorme. Então, a partir de agora, o que eu posso estudar?

  • Aprender sobre HTML e CSS;
  • Estude muito sobre Functions, ainda há muito sobre funções que você precisa aprender, que são um pouco mais avançados;

  • AJAX, quando você vai rolando a tela no yt e vai carregando mais conteúdo, isso é graças ao AJAX, uma tecnologia que usa JS;
  • NodeJS, além de ver apenas pelo lado do cliente vale a pena também dar uma estudada em NodeJS, para poder fazer seus scripts rodarem no seu servidor, isso dá um potencial muito grande para a linguagem;
  • Frameworks/Bibliotecas, como JQuerry, Angular, ViewJS, etc...

Objeto, o básico

Para declarar um array basta você declarar uma variável e colocar colchetes nela, tornando ela um vetor. O array te permite inserir diversos valores nele, você pode misturar nele string com números, etc. No entanto, você não consegue alterar o Índice (posição), esse índices numéricos podem te atrapalhar no futuro. E é aí que vamos um pouco mais fundo...

Para declarar um objeto basta, ao invés de colchetes você usar chaves{}, você pode dividir ele da mesma forma que o array, mas veja abaixo:


        let amigo = {nome:'josé'}

No pedaço de código acima você está nomeando o índice como nome, e inserindo o valor 'josé' nele.

Ou seja, com o objeto você tem mais possibilidades, mais liberdade e controle, do que com o array. Ele também tem uma funcionalidade especial, veja:


        let amigo = {engordar(p){}}

Você criou o "engordar" com o parametro "p" e abrindo/fechando cahves ({}), o que você está fazendo aqui, na verdade, é criar o item engordar que tem uma função dentro dele. Veja um exemplo: (prox-passos.js)

Se você usar um typeof tanto em um array quanto em object, ambos serão identificados como object. Isso ocorre por elas serem estruturas da mesma classe, tiveram a mesma origem.
