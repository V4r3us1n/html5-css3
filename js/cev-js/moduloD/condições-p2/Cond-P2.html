Condições (Parte 2)

Não existem apenas condições onde você ou faz uma coisa ou outra. Existem também situações em que existem várias outras possibilidades.

Mas é possível mesclar essas coisas veja um exemplo: 


        if (cond1){
            [bloco1]
        } else {
            if (cond2){
                [bloco2]
            } else {
                [bloco3]
            }
        }

É possível ver no exemplo acima, condições dentro de outra condições, é uma prática muito utilizado pelos programadores que é chamada de aninhamento e é feito dando um tab no código a ser aninhado.

Isso é tão comum que o próprio JS tem um comando especial chamado "elseif".

Além das Condições Aninhadas existem também as Condições Múltiplas que serve para valores fixos! Ela não serve muito intervalo de valores (ex: entre 1 e 5).

Ela te não te limita apenas a "sim" e "não", ela fornece vários valores fixos. Diferente do "if/else" ela não serve para todo tipo de situação, mas ela é muito útil para situações específicas.

Existem um comando no JS que se chama switch que funciona da seguinte forma: 


        switch (expressão) {
            case valor 1:
                [bloco1]
                break
            case valor 2:
                [bolco2]
                break
            case valor 3:
                [bloco3]
                break
            default:
                [bloco4]
                break
        }

Como visto acima o "switch" é uma expressão, onde cada opção dela é colocada em um case, no final de tudo tem o default que funciona da mesma forma que o "else". Vale ressaltar que após cada bloco tem que usar o comando break antes de ir para a próxima "case", no "default" ele é opcional.

O break serve para que, após o bloco ser executado o programa pule o switch e continue o resto do código, caso contrário ele vai entrar nas "cases" a baixo até encontrar o "break" ou chegar no final do "switch".
