Condições

A condição faz com que os códigos não sejam executados em sequência, existem bifurcações onde o comando pode ir por um caminho ou outro.

Uma das formas de representar essa condição com bifurcação é a seguinte: 


        if (condição) {
            condição = true
        } else {
            condição = false
        }

A estrutura possui dos blocos, um para caso a condição seja atendida e outro caso ela não seja.

Existem alguns tipos de condição, iremos começar abordando 2 deles...

Condição Simples

Possui apenas um tipos de bloco, que é para caso a condição seja atendida. Ex: 


        if (condição) {
            condição = true
        }

Nesse exemplo, caso a condição seja falsa ele vai seguir o código normalmente, como se a condição nem existisse.

Condição Composta

Possui os dois tipos de bloco, um para caso a condição seja atendida e outro caso ela não seja. Ex: 


        if (condição) {
            condição = true
        } else {
            condição = false
        }
