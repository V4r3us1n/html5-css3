Condições em JavaScript

Nesse módulo vamos tratar de todos os tipos de condições que existem em JS:

  • Condições Simples (if)
  • Condições Compostas (if/else)
  • Condições Aninhadas (if/else dentro de if/else dentro de if/else)
  • Condições Múltiplas (sweetcase)

Esse é o primeiro módulo em que existem exercícios propostos para se resolver.
