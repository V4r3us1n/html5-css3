Configurando iframes

Nem todos os navegadores para celular tem suporte para iframe, por isso é bom você colocar um comentário entre a abertura e fechamento do iframe, essa mensagem irá aparecer apenas se o navegador não for compatível!

Você pode botar no lugar da mensagem o link apra o site que você exibe no iframe.

Acessando site do guanabara,

O tamanho padrão de um iframe é de 300x150. Você pode alterar o tamanho e largura dele diretamente no html, apenas usando height="" e width="" ou você pode alterá-los no CSS.

Se você fazer alterações de altura e alrgura em um iframe tanto no HTML como no CSS, as alterações do CSS vão prevalecer!

Pode aparecer uma scroll-bar dentro do iframe, para modificá-la você pode usar o scrolling, que possuíalgumas variações:

  • 'auto': se o conteúdo for maior que o frame a barra de roalgem será gerada, caso contrário não.
  • 'yes': mesmo que o conteúdo coubesse apareceria a barra de rolagem.
  • 'no': não vai haver a barra de rolagem, só será mostrado o que couber no tamanho do iframe.

Só consegui usar o scrolling direto no HTML, no CSS isso não foi possível!

Em alguns navegadores alguns parâmetros podem não funcionar.

Existe outro parâmetro chamado frameboarder que aceita basicamente dois valores: 0 ou 1. POr mais que você tente colocar uma borda maior que 1px somente ela será gerada.

Você pode personalizar a borda do iframe usando um simples border ao invés do frameborder.
