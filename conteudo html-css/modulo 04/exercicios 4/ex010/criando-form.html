Criando o Primeiro Formulário

O que é um Formulário?

É um local onde você pode colocar dados.

Como funcionam os Formulários Online

Quando você quer comprar algo numa loja online, como shopee ou mercado livre, você tem que prencher algumas informações suas, essas informações são os dados que eles necessitam para concluir a operação.

Até mesmo quando você vai criar uma conta em uma rede social você tem que preencher um formulário.

Meu Primeiro Formulário

Todo formulário tem que estar dentro da pag "<form></form>".

Lembre-se que nem sempre porque você fez funcionar que essa é a melhor forma de fazer, como o exemplo que vamos criar agora.

Para criar a caixa onde o usuário vai inserir seus dados você usa o input

Claro, todo formulário precisa ter um botão para enviar os dados inseridos pelo usuário. Para criar ele você precisa usar o <input type="submit" value="Enviar"></input>, você coloca em value o texto que você quer que apareça no botão de envio.

Se você apertar em "Enviar" vai ver que a página vai caregar e vai permanecer igual, mas a pergunta é: para onde foram esses dados?

Se você reparar vai ver que não existe uma conexão entre a palavra "Nome" e a caixa que está ao seu lado, não existe uma ligação entre eles. Essa ligação é importante para que os mecanismos de busca saibam que existe uma ligação entre eles.

É esse tipo de configuração que teremos que fazer para tornar nosso formulário útil.
