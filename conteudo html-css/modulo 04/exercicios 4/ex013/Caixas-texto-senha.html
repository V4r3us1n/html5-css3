Como já vimos na aula anterior, as informações enviadas pelo formulário acima não tem proteção alguma, sendo fácil extrair essas informações.

Outros erro notável é que o formulário está sendo enviado mesmo sem ter nenhuma de suas área preenchidas.

Inputs do tipo reset servem para limpar as caixas de texto do formulário.

No entanto, após enviar o formulário é possível ver as informações inseridas no mesmo através da URL do site. Por isso lembre-se sempre de usar no form o method="POST".

Mas ainda da de contornar essa pequena barreira ao inspecionar as informações do site, para isso é necessário aprender HTTPS.

Como colocar atributos OBRIGATÓRIOS para o envio do fromulário

Para definir um campo do formulário como obrigatório basta você adicionar ao input correspondente required. Veja:

Se você não preencher um dos campos obrigatórios irá aparecer uma mensagem preparada pelo próprio navegador avisando que você precisa preencher esse campo para enviar o formulário.

Você pode definir outros requisitos como tamanho mínimo do que deve ser inserido em determinado campo (minlength="") ou caso você queira definir um valor máximo (maxlength="")

OBS: é sempre bom você definir minlength e maxlength dos campos do seu formulário.

Você pode alterar o tamanho da caixa do input usando size="" o valor inserido equivale a quantas letras cabem na caixa, podem ser digitadas mais letras, mas não vai aparecer todas elas.

Caso você queira inserir dicas do que preencher no campo você usa o placeholder="", veja no exemplo acima.

Para tirar o auto-completar basta usar o autocomplete="off". O autocomplete dentro do formulário só pode receber os valores on e off, já no input ele possui muitas outras possibilidades. Para colocar autocomplete no input tem que ter o autocomplete ligado no form.
