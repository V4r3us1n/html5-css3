Inconvenientes do iframe

Ele tem um problema com mecanismos de busca, existe um problema para o google bot acessar os conteúdos em iframe, assim ele não pode posicionar seu site da forma que ele deveria, dando a importância que ele merece.

O iframe também tem problema com Usabilidade e Acessibilidade, pessoas cegas (Acessibilidade), por exemplo, podem usar o computador através de Softwares Leitores de Tela, no entanto esses mesmos softwares apresentam problema para acessar os iframes.

Já na questão de Usabilidade, as vezes o próprio navegador ou o usuário podem acabar se confundindo com os iframes. Um exemplo disso seriam vc acessar outro conteúdo dentro do iframe e querer voltar para o inicial, clicando pra retroceder na página, isso faz com que alguns navegadores se confundam se é para retroceder a página ou o iframe.

Se voce^clicar para abrir uma nova janela alguns navegadores podem confundir se é para abrir essa janela no navegador ou no iframe. Além disso o site pode aparecer todo quebrado dentro do iframe caso ele não seja responsivo.

Também tem a questão de Segurança, os iframes, querendo ou não, acabam abrindo uma porta do seu site para outros sites. Você tem que tomar cuidado com o site para o qual você vai abrir essa porta, caso contráro você pode sofrer ataques, roubo de dados, coisas do tipo.
