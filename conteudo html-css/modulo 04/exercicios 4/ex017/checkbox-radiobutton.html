Checkbox e Radio button em HTML

O input do tipo Checkbox é usado em áreas onde você pode selecionar vários tópicos diferente, como no o que foi usado no exemplo abaixo.

Já os inputs do tipo Radio são usados em situações onde existem várias opções mas você só pode selecionar uma delas, como gênero.

No entanto para que você possa selecionar apenas um você deve deixar todos os inputs do tipo radio com o mesmo name, só o id deles que tem que ser diferente.

Em inputs do tipo radio recomenda-se adicionar valores para cada input, para esse valor ser enviado junto com o formulário. No formulário abaixo os gêneros "Feminino" e "Masculino" receberam os valores fem e masc, respectivamente.

Para deixar um dos input do tipo radio já marcado quando a página carregar, basta adicionar a um deles a propriedade checked. Essa propriedade também pode ser usada em inputs do tipo checkbox.

Você consegue deixar mais de um input do tipo checkbox marcado, quando a página carrega, utilizando a propriedade checked neles. No entanto, caso tente colocar essa propriedade em inputs do tipo radio apenas um deles ficará selecionado quando a página carregar, veja no exemplo abaixo.
