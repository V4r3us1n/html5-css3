Usar Label vai Melhorar seus Formulários

OBS: caso você queira tirar a opção de auto completar basta você colocar na tag form autocomplete="off"

Temos Problemas a Resolver

  • O formulário não está indo para lugar nenhum
  • Ligar os campos aos seus nomes

Para onde enviamos os dados?

Normalmente eles são enviados para alguma linguagem de programação, como java, php, etc. Vamos usar o PHP como exemplo.

Para enviar ele você adiciona na tag form action="cadastro.php", onde cadastro.php é um script feito separadamente para cadastrar os dados.

Obs: se você tentar enviar vai dizer que o arquivo não foi encontrado, isso é poruqe não temos esse script, ao menos não ainda, quem sabe no futuro... :P


Fazendo a Ligação entre Campos

Para ficar fácil diferenciar quando estamos usando Id ou Name, vamos colocar no em cada Id um "I" na antes da palavra.

Para fazer a ligação precisamos utilizar ETIQUETAS, também conhecidas como Label, para usar o Label basta você

Seleciona o texto que você quer ligar ao input e coloca ele dentro de um label, ao fazer isso ele vai pedir um parâmetro, o for, que é o que você vai usar (id ou name/p>

Obs: quando trabalhamos com php e html vamos utilizar mais o name, já no js é usado mais o id para fazer essa ligação. Basta você colocar nele o id da palavra que você quer que esteja relacionada com ele.


Após essas alterações você vai ver que visualmente ele ficou igual, mas fazer isso é extremamente importante. Uma detalhe importante é que agora, sempre que você clicar no nome que está ralacionado ao input o mesmo será selecionado! Isso é muito bom para dispositivos como celulares, onde a tela é pequena.

Sempre use Label em seus Formulários!

Exemplo

Quando você enviar o formulário, se você olhar onde é exibido o endereço que você está, você verá os seus dados ali, veja:

cadastro.php?nome=Vexx&sobrenome=xxeV

A interroção que aparece acima divide o que é o endereço do arquivo que foi aberto e os parâmetros que foram enviados


Tá, mas e se pedirmos usuário e senha, ela será exibida ali em cima também?
