A técnica mais versátil: CSS external style

Clique aqui para voltar à página anterior

Manter as folhas de estilo fora do código HTML, além de mais organizado, faz com que tudo seja reaproveitado de forma eficiente nas outras páginas do nosso site. Para isso utilizamos a tag <link> especialmente configurada para trabalhar com arquivos externos de estilo. Essa tag teve ser colocada na área <head> do documento. Ex: 


            <link rel="stylesheet" href="style.css">

Essa tag pode estar em qualquer linha, desde que esteja na área <head>, mas recomento que deixe ela após o <title>. Após criar essa tag clique em style.css e siga o link, isso criará um arquivo separado chamado # style.css.

Agora basta apenas você colocar todos as alterações que deseja fazer com CSS nesse novo arquivo :)

Ex: 


            @charset "UTF-8";
            /* Regra @charset caso hava problemas com acentuação*/

            body {
                background-color: chocolate;
                color: antiquewhite;
                font-size: 14pt;
                font-family: Arial;
            }

            h1{
                color:chartreuse;
            }

            a {
                color: aqua;
                text-decoration: none;
            }

Na linha 1 colocamos uma regra em CSS, que indica a compatibilidade com o padrão UTF-8, não é tão necessária, mas recomendo colocar. As regras são colocadas usando o "@" e elas ficam no início do arquivo.

Na linha 2 foi colocado apenas um comentário, que serve para explicar o motivo da linha 1.

Nas outras linhas foram feitas as configurações dos seletores.

Qual técnica eu escolho pra usar?

Nesse capítulo vimos 3 técnicas de uso da CSS: inline, interna e externa. Mas quando devo usar cada?

A técnica CSS inline style deve ser evitada, ela deixa seu código confuso. Só use ela em configurações muito pontuais e que não serão usadas em nenhum outro momento fora aquele.

Já a CSS internal style deve ser usada quando você for criar páginas isoladas, e somente quando as configurações de estilo for pequenas/médias, caso contrário vai deixar seu arquivo html muito grande, dificultando manutenções futuras.

Opte sempre pela CSS external style sempre que seu estilo for usado em várias páginas no seu site. Usando a tag <link> em várias páginas você pode compartilhar o mesmo estilo entre elas, sem precisar ficar alterando vários arquivos quando lhe for solicitado a alteração de uma cor, por exemplo.

De forma resumida:

  • CSS externo: use sempre que puder
  • CSS interno: use para pequenas configurações
  • CSS inline: procure evitar

Ainda é possível misturar as 3 técnicas, criando um CSS externo para as configurações globais, CSS interno para as configurações locais de um documento e CSS inline para pequenas configurações pontuais.

Alguns comandos de css: 


            h1 {
                color: blue;
                font-family: Arial;
                background-color: red;
                font-size: 40px;
                text-align: justify;
            }
