CSS: Introdução

Se o conteúdo de seu site não for bem apresentado seu alcance diminuirá. Visitantes de sites gostam de beleza, mantenha isso em mente.
É muito satisfatório ver um conteúdo organizado e bonito.

Você se lembra do que falamos sobre as CSS?

As CSS são as folhas em estilo cascata, todo e qualquer efeito visual que você queira colocar em seu site é feito através das CSS.

A forma mais simples de aplicar estilos: CSS inline style

Para fazer a técnica mais básica, para aplicar estilos em seu site, basta você ir na área <body> e criar o código a baixo: 

            
                <body style="font-family: Arial, Helvetica, sans-serif; color: rgb(47, 0, 255);">

Com o código acima foi alterado o formato da letra (de Times para Arial) e a cor da fonte foi alterada para roxo.

Caso você queira alterar apenas a cor do primeiro título use o seguinte código: be 


            <h1 style="color: red;">Capítulo 1<h1>

Observe

Apenas o h1 selecionado teve sua cor alterado, esse acima permaneceu igual ao resto. O motivo disso é porque fizemos uma configuração pontual usando CSS.

Estilizando de maneira mais interessante: CSS internal style

Para aplicar estilos de forma de maneira mais prática, podemos adicionar uma tag <style> dentro da área <head> de nosso documento. Ex: 


                <title>Introdução às CSS</title>
                <style>
                    body {
                        font-family: Arial,Helvetica, sans-serif;
                        background-color: black;
                        color: blue;
                    }
                    h1 {
                        color: red;
                    }
                </style>

Após essa configuração você vai se questionar: "se eu coloquei o h1 da cor vermelha porque o segundo h1 ainda está verde?"
Isso acontece porque as configurações pontuais vão prevalecer sobre as configurações gerais no CSS style
