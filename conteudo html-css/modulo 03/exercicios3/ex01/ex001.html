O que são Git e Github

Repositório Local/Remoto

Repositórios locais são aqueles que se encontram em seu dispositivo.

As vezes quando você vai tentar fazer sua versão de um projeto, você pode esquecer de deixar uma cópia de segurança, fazendo com que você perca o arquivo anterior.

Para evitar isso temos o Gerenciador de Versões, o git, um software que vai manter versões de seu projeto em seu computador. No entanto, se seu computador estragar você perde seus arquivos. Outro problema é que fica um pouco difícil mostrar seu projeto para outra pessoa quando vocês não estão em sua casa, certo?

Para isso existe o GitHub, um depende do outro, o Git guarda versões do seu projeto (Repositório Local), e o GitHub hospeda seus projetos na nuvem. Sem contar que mais de uma pessoa pode editar seu projeto.
