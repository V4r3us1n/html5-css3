Personalizando Seletores

Para criar estilos personalizados usamos o seletor de id (#) (Só pode ter um id na página, é uma regra) e de class (.) de maneira eficiente. No HTML podemos identificar um determinado elemento com um id único, você também pode agrupar elementos semelhantes com um class. Exemplos de propriedades classificadores: 


            <body>
                <h1 id="titulo-principal">Aprenda Desenvolvimento Web</h1>
                <h1>Aprenda HRML</h1>
                <h2 class="topico">Semântica Web</h2>
                <p class="texto">Lorem ipsum, ...?</p>
                <h1>Aprenda CSS</h1>
                <h2 class="topico">Estilos Web</h2>
                <p>Lorem ipsum dolor sit, ...ducimus.</p>

Na primeira linha vimos que o h1 principal possuí um id diferente dos outros.

Nas linhas 3 e 6 consuimos ver que ambos os h2 possuem a mesma class.

Um id vai identificar um elemento único dentro da página atual, permitindo criar estilos especiais para o elemento isolado.

Uma class vai identificar uma classe que pode haver um ou mais elementos pertencentes, compartilhando características em comum entre eles.

OBS: Só pode usar um id para um único elemento, não podemos ter mais elementos com um mesmo id dentro de uma mesma página. Mas podemos atribuir uma class para vários elementos que possuam essa mesma característica.

Você pode ver que mesmo que eu tenha aplicado um id no primeiro h1, ele continua igual aos demais, para alterar elementos com id ou class faça o seguinte: 


            #titulo-principal {
                background-color: darkgreen;
                color: white;
                text-align: center;
            }

            .topico {
                text-align: right;
            }

            .texto {
                text-align: justify;
            }

Anteriormente, na primeira linha, nós criamos um seletor personalizado para identificar o h1 principal como titulo-principal. Para fazer modificações em id colamos primeiro o símbolo # em seguida do nome único atribuído ao elemento.

Nas linhas 7 e 11 estamos fazendo alterações nas classes recebidas pelas linhas 3, 4 e 6 do exemplo anterior. Para fazer alterações em uma class usamos o símbolo . seguido do nome da classe que se deseja alterar.

Propriedades herdadas

Dar um id fará com que as alterações feitas nos demais não influencie os resultados no elemento, caso essas características conflitem entre si. Ou seja, as alterações no id se sobressaem sobre as alterações feitas para todos.

Pseudo-classes e pseudo-elementos

Uma pseudo-class CSS é uma palavra-chave adicionada às declarações de um seletor após um sinal de dois pontos e especificam um estado especial de um elemento. Existem várias pseudo-classes para estilos, alguns exemplos são: :hover, :visited, :active, :checked, :empty e :focus.

Um pseudo-elemento CSS é uma palavra-chave adicionada às delcarações de um seletor após dois sinais de dois pontos, permitindo que você formate um pedaço específico do elemento referenciado, elas estão relacionados ao estado do elemento atual, se eles está ativo, vazio, etc. Os principais pseudo-elementos usados são: ::before, ::after, ::first-letter, ::first-line.
