Ainda podemos usar tabelas em HTML?

Muitas pessoas dizem que tabelas são inúteis para um site, mas isso não é verdade, veremos adiante qual a função de uma tabela em um site!

Já vou deixar bem claro uma coisa aqui: VOCÊ NÃO PODE USAR TABELA PARA CRIAR A ESTRUTURA DO SEU SITE.

Você pode usar tabelas para apresentar dados, por exemplo, nome de países, habitantes, bandeira, etc...

Sua primeira tabela em HTML

A primeira coisa que precisamos fazer é criar uma tag <table>, que vai delimitar a sua tabela.

Vamos ver a Hierarquia de uma Tabela Simples:

  • table (tabela)
    • [tr] table row (linha da tabela)
      • [th] table header (cabaçalho da tabela)
      • [td] table data (dados da tabela)

Exemplo de tabela (essa não tem cabeçalho)

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4

OBSERVAÇÃO: o W3C diz que é opcional o fechamento das tags <tr> e <td>. Além disso, vale a pena citar que se a hierarquia não for respeitada os dados podem aparecer ou antes ou depois da tabela, e não dentro da mesma.

Originalmente ela não se parece tanto com uma tabela, para resolver isso vamos colocar um pouco de estilo nela!

DICA: você pode usar o "border-collapse: collapse;" para unir as bordas das células, porque, caso você coloque uma borda elas vão ter um espaçamento entre elas.

Alinhando o conteúdo em Tabelas

Vamos para um próximo exemplo, dessa fez vamos aumentar também a altura da tabela...

A1 B1 C1
A2 B2 C2
A3 B3 C3
A4 B4 C4

Para fazer um alinhamento horizontal basta usar o "text-align: center;".

Para fazer o alinhamento vertical basta usar o "vertical-align: ;", onde as opções são "top", "bottom" e "middle".
