O github sempre mostra quais repositórios estão ativos na máquina, você pode adicionar mais sempre que quiser.

Essa alteração feita aqui é notificada no repositório local e precisamos fazer um commit no github desktop, para salvar essa nova versão.

Para passar essa nova versão do repositório local para o web é necessário você clicar na opção "Push Origin".

Colocando o site no ar

Vamos usar o GitHub Pages para poder hospedar gratuitamente nosso site com ajuda do github.

Primeiro você vai abrir o repositório que você deseja hospedar gratuitamente para poder mostrar para os coleguinhas.

Procure o arquivo/pasta que você deseja hospedar dentro desse repositório.

Clique na opção "Settings" que é exibida abaixo do nome do repositório e de suas avaliações, etc.

Procure por "GitHub Pages", clique nele. Após isso clique na opção "Choose a theme" e selecione o tema que preferir, clique em "Select Theme" após fazer a escolha.

Agora você tem o endereço de sua hospedagem, que no meu caso foi: https://v4r3us1n.github.io/html5-css3/. Para ver o link de sua hospedagem basta ir novamente em "GiHub Pages".

O que será mostrado nesse link será o ".README" que foi criado anteriormente. Para você poder acessar os arquivos por esse link basta você colcar os links de cada exercício no ".README"
