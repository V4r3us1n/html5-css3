Uso de Funções em JS

Nós já aprendemso a criar e usar funções simples, mas está na hora de subir de nível, ainda temos que falar sobre: passagem de parâmetros e retorno de resultados.

O JS é uma linguagem que funcional sobre um paradigma funcional, por isso que função, em JS, é algo muito importante.

Imagine a seguinte situação:

  1. Sua mãe te chama e te dá uma ordem, como ir comprar eleite (chamada, não precisa ser algo dito, pode ser por um evento: se "chover" recolha a roupa do varal)
  2. Sua mãe te dá dinheiro para você poder comprar leite (dinheiro, o que sua mãe quer == parâmetro)
  3. Você vai no emrcado, compra o leite e volta (ação)
  4. Você chega em casa e entrega o leite para sua mãe (retorno)

Ou seja, uma função pode ter os seguintes fatores: chamada, conjunto de parâmetros, ação e retorno. Nem toda função usa parâmetros ou possuí retorno, mas isso dá mais funcionalidades a sua tarefa.

Aqui a definição de uma função:

"Funções são ações executadas assim que são chamadas ou em decorrência de algum evento."

"Uma função pode receber parâmetros e retornar um resultado."

Para você ver como é na prática, aqui está um modelo de função:


        function ação(param) {
            [bloco]
            return res
        }

        ação(5)

No modelo acima você criou a função ação e, posteriormente, chamou ela (ação(5)) utilizando como parâmetro o número 5. Após executar a função utilizando o parâmetro solicitado será enviado o resultado da função dentro da variável res (return res)

Vamos ver uma função de verdade, para ver como funciona na prática:


        function parimp(n) {
            if (n % 2 == 0) {
                return "PAR";
            } else {
                return "IMPAR";
            }
        }

        let res = parimp(11);

Uma função em JS só pode ter um return, a função de cima tem dois porquê eles estão dentro de if/else, ou seja, apenas um deles será retornado!

Veja a função acima sendo posta em prática em "Funcoes2-JS.js"

Se você criar uma função que exija dois parâmetros mas apenas um deles seja fornecido a função não irá funcionar direito. Pense que é uma função de soma, com apenas um parâmetro vai ficar n1 + undefined = NaN, e não tem como usarmos isso.

Uma solução para esse problema é preestabelecer valores para caso não seja passado um valor para os parâmetros, você faz isso da seguinte forma:


        function soma(n1=0, n2=0) {
            return n1 + n2;
        }

        let s = soma(1);
        console.log(s);

Caso seja passado um novo parâmetro o n1 e n2 deixam de possuir valor 0!

Outra coisa que você pode fazer é colocar uma função dentro de uma variável, isso é possível no JS. Veja um exemplo:


        let v = function(x) {
            return x*2;
        }
        
        console.log(v(5));

Nesse caso, você passa um parâmetro para a variável e ela executará uma função, nesse caso ela retornará como resposta o dobro de 5, ou seja, 10.

Existem muitas outras sintaxes de funções no JS, mas esse curso é apenas sobre o básico de JS, então não vamos entrar mais a fundo!
