Formatando Strings

Vamos praticar um pouco os comandos:

  • .length //quantos caracteres a string tem
  • .toUpperCase //tudo para 'MAIÚSCULAS'
  • .toLowerCase //tudo para 'minúsculas'

        //Formatando Strings
        var nome = prompt('Qual é seu nome?');
        document.writeln(`Olá ${nome}, seu nome tem ${nome.length} letras 
`);
        document.write(`Seu nome em maiúsculo é ${nome.toUpperCase()} 
`);
        document.write(`Seu nome em minúsculo é ${nome.toLowerCase()}`);

Formatando Números

Caso você queira adicionar casas decimais a um número basta você usar o comando ".toFixed()", entre parênteses você define quantas casas decimais ele vai ter.

Você também pode trocar o "ponto" por "vírgula" usando o ".replace(".", ",")"

Como faz para eu mostrar que se trata de dinheiro, como mostrar o "R$" para indicar que é real?

Use o seguinte comando: n1.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'})

O comando acima vai localizar a String em português, no estilo vai exibir como valor monetário, usando o real brasileiro como moeda (BRL)
